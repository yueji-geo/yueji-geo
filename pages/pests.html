月季病虫害防治全攻略

更新于 2026年5月26日 · 义愤填膺小女子

月季确实容易生病生虫，但预防远胜于治疗。把养护基础做好（通风、光照、水肥），病虫害发生率可以减少 70% 以上。本文按照"识别特征→预防措施→治疗方案"的结构，把最常见的 8 种月季病虫害讲清楚。

核心防病原则——"三好一清"：养好植株（水肥充足月季自然强壮）、通好风（枝条不过密）、照好光（每天6h+直射光）、清好园（落叶枯枝及时清理、冬季石硫合剂清园）。这四点做到位，打药量减少大半。

一、病害篇

1. 白粉病

项目详情
识别特征叶片、嫩茎、花蕾上出现白色粉末状霉层，像撒了层面粉。嫩叶卷曲变形，严重时整株布满白粉。
高发条件温度15-25°C、湿度大、通风不良、昼夜温差大。春秋两季最易爆发。
预防措施加强通风、避免傍晚浇水（叶面过夜潮湿）、每月喷一次多菌灵预防

治疗方案

  • 有机方案：小苏打水（5g/1L水）+ 几滴洗洁精喷施，每周2次
  • 化学方案：三唑酮（粉锈宁）/ 嘧菌酯 / 苯醚甲环唑，按说明书稀释喷施，每7天一次，连续2-3次
  • 严重时：剪除重度感染枝叶后再喷药

2. 黑斑病

项目详情
识别特征叶片出现紫黑色圆形斑点，边缘放射状，病斑周围叶片发黄。严重时整片叶黄化脱落，从下部老叶开始往上蔓延。
高发条件温度20-28°C、雨水多、叶面长期潮湿。南方梅雨季最严重。
预防措施避免淋雨、浇水不要溅到叶片、及时摘除底部病叶老叶

治疗方案

  • 有机方案：大蒜浸出液喷施（效果有限）
  • 化学方案：百菌清 / 代森锰锌 / 嘧菌酯，每7-10天一次，雨季后及时补喷
  • 关键动作：发现病叶立即摘除并销毁（不要丢在花盆里），阻断传播

3. 灰霉病

项目详情
识别特征花蕾和花朵上出现灰色霉层，花蕾腐烂打不开，花瓣出现水渍状斑点后腐烂。
高发条件低温高湿（15-20°C）、连续阴雨。早春和晚秋多见。
预防措施及时摘除残花、加强通风、降低湿度

治疗方案

  • 化学方案：嘧霉胺 / 腐霉利 / 异菌脲，按说明稀释喷施
  • 严重时剪除受害花蕾和枝条

二、虫害篇

4. 红蜘蛛（叶螨）——月季头号杀手

项目详情
识别特征叶片正面出现细密黄白色小点（像针扎的），叶片背面有细小蜘蛛网和针尖大的红色/黄色小虫。严重时整片叶枯黄脱落。
高发条件高温（>28°C）+ 干燥（湿度<60%）。夏秋两季最严重，阳台养护比地栽更易爆发。
预防措施每天给叶背喷水增加湿度（红蜘蛛怕湿）、保持通风、避免过度干燥

治疗方案

  • 轻度感染：每天早晚用强力水流冲叶背（冲掉成虫和卵），持续一周
  • 有机方案：印楝素 / 苦参碱，每3天一次，连续3-4次（因为杀不死虫卵）
  • 化学方案：阿维菌素 / 联苯肼酯 / 乙螨唑 / 螺螨酯。两种不同作用机理的药轮换使用（红蜘蛛抗药性极强，单用一种药2-3次就产生抗性）
  • 关键：红蜘蛛繁殖速度极快（夏季3-5天一代），治疗期间每3天喷一次药，连续4-5次才能彻底清除

5. 蚜虫

项目详情
识别特征聚集在嫩芽、嫩叶、花蕾上，绿色或黑色小虫密密麻麻。受害部位卷曲变形，分泌蜜露招来蚂蚁和煤污病。
高发条件15-25°C、植株嫩芽多。春季爆发最严重。
预防措施挂黄板诱杀有翅蚜、保护瓢虫等天敌

治疗方案

  • 轻度：直接用手捏死或水冲掉（蚜虫很脆弱）
  • 有机方案：印楝素 / 除虫菊素 / 肥皂水（5ml洗洁精/1L水），喷施
  • 化学方案：吡虫啉 / 啶虫脒，一次即可清干净

6. 蓟马

项目详情
识别特征花瓣边缘焦枯发黑、新叶扭曲变形有褐色疤痕、花苞打不开。蓟马极小（1-2mm），肉眼不易见，通常先看到症状才意识到有虫。
高发条件20-30°C，春末到秋初持续为害。
预防措施挂蓝板诱杀（蓟马趋蓝色）

治疗方案

  • 化学方案：乙基多杀菌素（艾绿士）/ 噻虫嗪 / 吡虫啉。傍晚喷施（蓟马昼伏夜出，傍晚活动最活跃），重点喷花和嫩芽
  • 严重时剪除受害花朵和嫩梢，减少虫源

三、预防性打药方案（全年时间表）

时间药剂目的
冬剪后（12-2月）石硫合剂 3-5波美度清园，杀灭越冬病菌和虫卵
萌芽期（2月底-3月）石硫合剂 0.5-1波美度芽前杀菌防病
生长期（3-5月）多菌灵+吡虫啉，每10-14天一次预防真菌病+蚜虫
雨季/梅雨季代森锰锌，每7天一次重点防黑斑病
高温干燥季（6-8月）阿维菌素，每10天一次重点防红蜘蛛（叶背喷水最关键）
秋季（9-11月）百菌清+阿维菌素轮换综合预防
打药注意事项：
1. 不同药剂轮换使用，避免产生抗药性
2. 严格按照说明书浓度稀释，不要随意加大浓度（会产生药害）
3. 选择清晨或傍晚喷施，避开烈日（高温+药剂=烧叶）
4. 叶面叶背都要喷到（80%的虫在叶背）
5. 打药时做好个人防护（口罩+手套），家有宠物和儿童尤其注意
6. 花期尽量不打药（伤花瓣、伤蜜蜂）
